# 充值功能

## 获取充值地址（风）

api/v1/account/depositAddress

GET

**请求参数**

| **参数**  | **是否必须** | **类型** | **说明**           |
| --------- | ------------ | -------- |------------------|
| accountId | true         |          | 账户id，具有唯一性 |
| network   | true         |          | 转账网络          |
      
**返回参数**

| **参数**     | **类型** | **说明** |
| ------------ | -------- | -------- |
| chainAddress |          | 地址     |
